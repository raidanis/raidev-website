## 📝 How to Use

### Quick Start
```bash
# 1. Navigate to project
cd d:\RAIdev

# 2. Run setup (Windows)
setup.bat
# or (Mac/Linux)
./setup.sh

# 3. Start server
php artisan serve

# 4. Open browser
# http://localhost:8000
```

### Customization
1. **Edit Pages**: `resources/views/*.blade.php`
2. **Change Styles**: `resources/css/style.css`
3. **Update Navigation**: `resources/views/layouts/app.blade.php`
4. **Add Routes**: `routes/web.php`
5. **Add Scripts**: `resources/js/main.js`

### Deployment
1. Update `.env` for production
2. Run optimization commands
3. Deploy to hosting platform
4. Configure email (optional)
5. Point domain name
